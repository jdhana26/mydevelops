Bootstrap

Bootstrap is a free, open-source front-end framework for building responsive and mobile-first websites and web applications. It provides a collection of pre-designed templates, CSS styles, and JavaScript components to help developers create visually appealing and consistent web interfaces efficiently.

BreakPoint

Bootstrap uses a mobile-first approach with predefined breakpoints for responsive design. Adjusts layout automatically based on device width.

Classes:

  • Small - sm
  • Medium - md,
  • Large - lg,
  • Extra Large - xl,
  • Extra Extra Large - xxl

Example:
< div class="col-12 col-md-6 col-lg-4">

Container & Container Fluids

Bootstrap containers are used to center and pad your site’s contents. .container → Fixed-width container (responsive at each breakpoint) .container-fluid → Full-width container (100% width across all devices)

Classes:

  • Container Small -> container-sm
  • Container Medium -> container-md
  • Container Large -> container-lg
  • Container Extra Large -> container-xl
  • Container Extra Extra Large -> container-xxl
  • Container Fluid -> container-fluid

Typography

Bootstrap provides pre-styled text utilities for headings, alignment, and more.

Classes:

  • .h1 to .h6 - Heading sizes
  • display-1 to .display-6 - Larger heading styles
  • lead - Highlights introductory text
  • text-start, .text-center, .text-end - Text alignment
  • .fw-bold, .fw-light - Font weight
  • .fst-italic - Italic text
  • .text-uppercase, .text-lowercase, .text-capitalize - Text transformations

Images

Images in Bootstrap are made responsive with .img-fluid. This applies max-width: 100%; and height: auto; to the image so that it scales with the parent width.

Classes:

  • .img-fluid - Makes image responsive (max-width: 100%, height: auto)
  • .img-thumbnail - Adds a border and padding
  • .rounded, .rounded-circle, .rounded-0 - Controls border-radius
  • .float-start, .float-end, .mx-auto d-block - Positioning

Colors & Background Colors

Bootstrap provides contextual text and background color utilities.

Text-Colors:

.text-primary, .text-secondary, .text-success, .text-danger, .text-warning, .text-info, .text-light, .text-dark, .text-muted, .text-white

Background-Colors:

.bg-primary, .bg-secondary, .bg-success, .bg-danger, .bg-warning, .bg-info, .bg-light, .bg-dark, .bg-white, .bg-transparent

Flexbox Utilities

Classes:

  • .d-flex - Enables flexbox
  • .flex-row, .flex-column - Direction
  • .justify-content-start, .justify-content-center, .justify-content-between, .justify-content-end - Horizontal alignment
  • .align-items-start, .align-items-center, .align-items-end - Vertical alignment
  • .flex-wrap, .flex-nowrap - Wrapping behavior
  • .order-0 to .order-5 - Item order

Container Fluids - Class

  • Creates a full-width layout that spans the entire viewport width.
  • Often used for headers, footers, and sections requiring edge-to-edge backgrounds.
